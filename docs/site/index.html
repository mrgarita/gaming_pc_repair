PROJECT — 故障したゲーミングPCを復活させる

電源は入る、しかし起動しない。
1 台のゲーミングPCを復活させるまでの記録

Ryzen 7 3700X と RTX 2070 SUPER を積んだ G-Tune 製ゲーミングPCが、SSD 増設と BIOS 更新を経たある日、ゲーム中の暗転を最後に起動しなくなった。原因の切り分けから 部品交換、復活までの経過を、この備忘録に残していく。

MACHINE STATUS — 診断LED（現在の状態）

  • CPU
  • DRAM
  • VGA
  • BOOT

マザーボード交換後は一発起動し、Windows 11 の導入まで完了。高負荷での安定性検証（フェーズ3）はこれから（2026-07-22 時点）

進め方

修理は 4 つの step で進める。step3 の動作検証で動かなければ step1 の切り分けに 戻る——このループが本プロジェクトの骨格になる。

備忘録

  • STEP 1 2026-07-11 · UPD 2026-07-16

    現状把握 — 症状の経過と原因の切り分け

    構成の整理、SSD 増設から起動不能に至るまでの時系列、実施済みの切り分けと 「マザーボードが最有力」という現時点の見立てをまとめた。

  • STEP 2 2026-07-11 · UPD 2026-07-16

    部品調達 — AM4 マザーボードの選定と交換方針

    CPU を流用するための規格（AM4）とリテール品の解説、候補 2 枚（ASRock B550 PG4 / ASUS TUF B550-PLUS）の仕様比較と搭載計画、「B550 先行交換 → 直らなければ電源」の 二段構え、購入候補リスト、Windows 11 再認証の手順をまとめた。

  • STEP 3 2026-07-16 · UPD 2026-07-22

    組み立て・検証 — 新板で起動し、Windows 11 を導入

    部品到着直後に旧 OS SSD（CFD PG3VNF）の認識不良が発覚。OS を P310 へ クリーンインストールする計画修正、データ救出計画、マザーボード交換手順、 4 フェーズの動作検証チェックリストをまとめた。7/17 に旧マザーボード取り外し（作業記録①・写真 17 枚）、 7/20 に新板組み込み → BIOS 一発起動（作業記録②・写真 22 枚）、7/22 に Windows 11 をクリーンインストールしてデスクトップ到達——ただし ライセンス認証は OEM 移行不可でリテール版を購入して解決（作業記録③・写真 25 枚＋図4）。 フェーズ 1・2 通過。
