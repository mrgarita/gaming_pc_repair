部品調達 — AM4 マザーボードの選定と交換方針

予備部品がなく電源の事前切り分けができないため、「最有力容疑のマザーボードを 先に交換し、直らなければ電源」という二段構えで進める。CPU を流用するための 規格（AM4）の整理と、リテール品を選ぶ理由、購入候補、交換後の Windows 再認証手順をまとめる。

交換方針 — 二段構え

  1. 第 1 段：B550 リテールのマザーボードに交換する。step1 の切り分けで容疑が最も濃く、OEM マザーボード＋BIOS 起因の懸念も同時に解消できるため。CPU・メモリ・GPU・SSD・電源・ケースはすべて流用する。
  2. 第 2 段：直らなければ電源を交換する。次点の容疑。マザー交換後も CPU・DRAM LED 点灯 → 再起動ループが続く場合に購入する（先走って買わない）。

リスクの整理：電源が真犯人だった場合、第 1 段のマザー代が「先行投資」になる。 ただしその場合も OEM 板からリテール板への置き換え自体は無駄にならない （BIOS 更新・保証・将来の切り分けやすさが手に入る）。

なぜ「AM4」のマザーボードなのか

AM4 は AMD の CPU ソケット（CPU の差し込み口）の規格名。CPU とマザーボードは ソケット規格が一致しないと物理的に取り付けられない。手持ちの Ryzen 7 3700X は AM4 対応 CPU なので、CPU を流用するには同じ AM4 ソケットのマザーボードを選ぶ。 現行最新の Ryzen（7000〜9000 世代）は AM5 という別規格で、3700X は載らない。

チップセットは B550 を選ぶ

同じ AM4 でも、基板上の制御チップ「チップセット」でグレードが分かれる。 元の PC が X570 だからといって X570 に戻す必要はない。

X570上位（元の板と同じ）。3700X に対応するが高価で、2026 年現在は新品の入手性が悪い
B550中位・定番。今回の本命。3700X（Ryzen 3000 世代）を標準サポートし、新品が約 12,000〜15,000 円で流通中
A520入門。対応はするが拡張性が低く、価格差も小さいので選ぶ理由が薄い

購入前の確認：製品ページの「対応 CPU リスト」に Ryzen 7 3700X（Matisse）が あることだけ確認する。B550 は発売時から Ryzen 3000 対応のため、基本的に問題ない。

リテール品と OEM 版の違い

リテール品とは、一般小売向けに単品販売される正規パッケージ品のこと。 いまの PC に入っているのは「OEM 版」——メーカー（マウスコンピューター）が 自社 PC 組み込み用に調達した特注版で、BIOS の管理・提供はメーカー側にあり、 単品での入手や単品保証はできない。

リテール品に交換すると、(1) ASRock 等のメーカーから BIOS 更新が直接提供される、 (2) 店舗・代理店の単品保証が付く、(3) step1 で「考慮」に挙げた OEM マザーボード＋BIOS 起因の不具合という懸念ごと解消できる、という利点がある。

Windows 11 の再認証手順（マザー交換後）

OS は Windows 10 Home からの無償アップグレード版 Windows 11 で、 Microsoft アカウントに本機を紐付け済み。デジタルライセンスがアカウント側に 記録されているため、次の手順で再認証できる見込みが高い。

  1. マザーボード交換後、Windows 11 を起動してインターネットに接続する
  2. 「設定 → システム → ライセンス認証」を開く
  3. 認証エラーが出ている場合は「トラブルシューティング」を実行する
  4. 「このデバイスのハードウェアを最近変更しました」を選択する
  5. Microsoft アカウントでサインインし、デバイス一覧から本機を選んで再認証する

Microsoft は再認証を 100% 保証していないが、アカウント紐付けがある場合は 通るのが通例。通らなかった場合は画面の案内（サポート窓口）に従う。

任意：電源の簡易チェック（クリップテスト）

部品を買う前に無料でできる唯一の電源チェック。「最低限起動するか」だけを見る 簡易テストで、回った＝電源は正常、とまでは言えない点に注意 （負荷時だけ落ちる不良は検出できない）。

  1. 電源ケーブルをコンセントから抜き、電源ユニット背面のスイッチを OFF にする
  2. マザーボードから 24 ピンコネクタを外す
  3. 24 ピンコネクタの PS_ON（緑線）と COM（黒線）を、伸ばしたゼムクリップで短絡する
  4. 電源ケーブルを差し、背面スイッチを ON → 電源ファンが回るか確認する
  • ファンが回らない → 電源のクロ確定。第 2 段を繰り上げて電源から交換する
  • ファンが回る → 最低限は動く。ただしシロ確定ではないので方針は変えない

安全上の注意：作業は必ず通電を切ってから行い、電源ユニットの筐体は開けないこと （内部コンデンサは通電を切っても高電圧を保持している）。
