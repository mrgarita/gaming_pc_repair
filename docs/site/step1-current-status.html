現状把握 — 症状の経過と原因の切り分け

SSD 増設をきっかけに始まった不調は、BIOS 更新でいったん収まったのち、 ゲーム中の暗転を境に「起動しない」状態へ悪化した。ここまでに分かっていることを 構成・時系列・切り分け結果の順に整理する。

対象マシンの構成

ベースはマウスコンピューター G-Tune のミドルタワー機（購入時 Windows 10 Home）。 主要部品は次のとおり。

マザーボードASRock X570 Phantom Gaming 4 ベース（OEM 版）
BIOSP2.01 → P5.80 へ更新済み
CPUAMD Ryzen 7 3700X（8コア/16スレッド）
CPU クーラー空冷（ダイヤモンドグリス使用）・未着脱
GPUNVIDIA GeForce RTX 2070 SUPER 8GB
メモリDDR4-3200 16GB × 2（計 32GB）
SSD（既存）M.2 NVMe 1TB — Windows 入り（M2_1 スロット）
SSD（増設）Crucial P310 1TB — 今回の増設品
電源800W 80PLUS Titanium
OSWindows 11（購入時の Windows 10 Home から無償アップグレード）・Microsoft アカウント紐付け済み

症状の経過

LED は経過の状態を示す： 異常 ／ 対処・変化点 ／ 正常。

  1. 異常Crucial P310 を増設すると Windows が起動せず、VGA・BOOT LED が点灯
  2. 正常P310 を外すと正常起動
  3. 対処BIOS を P2.01 から P5.80 へ更新
  4. 正常P310 装着でも正常起動。Windows で SSD を認識し、初期化・フォーマット完了
  5. 異常Steam ゲーム中に画面が突然真っ黒になり再起動
  6. 異常以後、P310 を外しても起動しない
  7. 異常CPU・DRAM LED が同時点灯し、数秒後に再起動を繰り返す
  8. 異常CMOS クリア・メモリ 1 枚（A2）・SSD 全取り外しでも改善せず
  9. 対処マザー交換の準備中、旧 OS SSD をケースに入れて別 PC で中身の確認を試みる（2026-07-16）
  10. 異常読み込みが数分単位で停滞したのち認識消失——旧 OS SSD 自体の不調が発覚（詳細は step3）

増設直後の VGA・BOOT LED（経過 1）と、現在の CPU・DRAM LED（経過 7）は 別の症状である点に注意。前者は BIOS 更新で解消しており、後者が現在の本丸。

実施済みの切り分け

  • BIOS 更新（P2.01 → P5.80）
  • CMOS クリア
  • メモリ 1 枚ずつの確認（A2 スロット）
  • SSD 全取り外しでの起動確認
  • CPU・CPU クーラーの着脱とソケット目視（2026-07-11）— ピン曲がり・焼損なし

クーラーはグリスを再塗布せずそのまま再装着している（手持ちがないため）。起動確認 程度なら問題ないが、復活後に常用を始める前には塗り直しが必要。グリスは調達リスト入り。

切り分けマップ

ここまでの結果を部品ごとの容疑度として模式図に落とすと次のようになる。 メモリと SSD は外しても症状が再現するためシロ寄り。最小構成でも CPU・DRAM LED 点灯 → 再起動ループが続くことから、残る容疑はマザーボード・ 電源・CPU に絞られる。なお後日、旧 OS SSD 自体に別の不調が発覚した （2026-07-16）。「起動ループの原因としてはシロ寄り」の判定は変わらないが、 詳細は step3 に記録した。

現時点の見立て

  • 容疑◎ マザーボード故障。SSD・メモリを外した最小構成でも CPU・DRAM LED 点灯 → 再起動ループが続くため、共通基盤であるマザーボードが最有力。
  • 容疑○ 電源ユニット不良。発端が高負荷（ゲーム中）の暗転である点と整合する。次点。
  • 容疑△ CPU 故障。クーラーを外しての目視では異常なし（2026-07-11）。電気的な故障の可能性だけが残るが、可能性は低い。
  • 考慮 OEM マザーボードと BIOS（P5.80）の組み合わせ起因の不具合も考慮しておく。
  • 考慮 根本原因が電源で、マザーボードや SSD が巻き添えになった筋も考慮。旧 OS SSD の不調発覚（2026-07-16・step3）はこの筋とも整合しうる。

次のアクション（step2 へ）

  1. 無料でできる切り分けは消化済み。CPU クーラーの着脱と目視は実施し異常なし（2026-07-11）。予備部品がないため、電源の事前切り分けは不可と判明した。
  2. 交換方針は「B550 リテールのマザーボードを先行交換 → 直らなければ電源」の二段構え。規格（AM4）の解説・候補部品・Windows 再認証の手順は step2 部品調達 にまとめた。

Windows 11（Windows 10 からの無償アップグレード）は Microsoft アカウントに 紐付け済みと確認できた。マザーボード交換後の再認証手順は step2 のページに記載。
